# Heart Disease Prediction

## Project Overview
This project focuses on predicting the presence of heart disease using a Logistic Regression model.

Exploratory Data Analysis (EDA) was performed using visualization techniques before training the classification model.

---

## Technologies and Libraries
- Python
- Pandas
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Key Concepts Implemented
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Train-Test Split
- Logistic Regression
- Model Evaluation

---

## Model Evaluation
The model was evaluated using:

- Accuracy Score
- Precision Score

Accuracy measures overall correctness of predictions.  
Precision measures how many predicted positive cases were actually correct.

---

## Project Workflow
1. Data Loading  
2. Data Visualization  
3. Train-Test Split  
4. Model Training using Logistic Regression  
5. Performance Evaluation  

---

## Key Outcome
The Logistic Regression model successfully predicted heart disease cases, with evaluation metrics used to assess classification performance.
